# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-03-05

### Added

- 15 built-in prompt-injection detection rules.
- Two-layer scanning with an optional LLM detector for semantic attacks.
- Sanitizer with 4 modes: label, redact, escape, and block.
- Async scanning support across Python and TypeScript APIs.
- Middleware wrappers for tool and content scanning pipelines.
- Native integrations for OpenAI, Anthropic, and LangChain.
- Structured audit trail events and observability hooks.
- 100% test coverage across the initial release.
